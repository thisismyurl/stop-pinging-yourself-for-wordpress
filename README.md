# Stop Pinging Yourself

Stops a WordPress blog from sending pingbacks to itself when posts contain internal links. Eliminates the noise of self-pingback comments without disabling the pingback system entirely.

[![WordPress.org](https://img.shields.io/wordpress/plugin/installs/stop-pinging-yourself-for-wordpress.svg)](https://wordpress.org/plugins/stop-pinging-yourself-for-wordpress/)
[![Rating](https://img.shields.io/wordpress/plugin/r/stop-pinging-yourself-for-wordpress.svg)](https://wordpress.org/plugins/stop-pinging-yourself-for-wordpress/#reviews)

## Why this exists
WordPress sends a pingback every time a post links to another post — including self-links. The result is dozens of "pingback" comments cluttering the moderation queue with no value.

This plugin filters self-pings before they leave the site. External pingbacks continue to work as normal.

## Features
- Blocks only self-referential pingbacks.
- Zero configuration.
- 600+ active installs, 100% rating on WordPress.org.

## Status
Maintenance mode. The plugin is intentionally minimal and does not need active development.
